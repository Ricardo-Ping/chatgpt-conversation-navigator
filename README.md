# ChatGPT Branch Map

一个运行在 `chatgpt.com` / `chat.openai.com` 的 Chrome 扩展，用于把长对话变成可定位、可分支、可回看的结构化导航。

## 当前版本做什么

### 1. 双模式界面
- 轻量模式（默认）：`小眼睛 + 问题栏`，用于快速定位消息。
- 极简模式：`展开 / 顶部 / 底部 / 上 / 下` 的悬浮胶囊。

### 2. 消息扫描与导航
- `扫描` 会重新抓取当前会话消息。
- 扫描提示显示：`总消息数` + `提问数（按非空用户消息统计）`。
- 问题栏每一行按 `你说 + ChatGPT 子行` 分组展示。
- 支持搜索（含中文输入法 IME）。
- 支持顶部/底部快速跳转。
- `上 / 下` 只在“你的提问消息”之间跳转，且到边界会提示。

### 3. 中心跟随高亮（动画）
- 你滚动主页面时，问题栏会自动高亮“当前视口中心对应提问”。
- 高亮带平滑阴影和过渡动画，方便随读定位。

### 4. 设为节点 / 取消节点
- 只在 ChatGPT 回复消息旁显示 `设为节点`。
- 点击后变为 `已设节点` 并常驻显示。
- 再次点击 `已设节点` 可取消节点。
- 问题栏中 ChatGPT 子行会显示节点标识（`✦`）。

### 5. 开分支（原生入口 + 节点选择）
- 点击 `开分支` 时优先从“已设节点”中选择。
- 若存在多个已设节点，会弹出动画菜单让你选择分支起点。
- 菜单编号与问题栏编号对齐（不是从 1 重新排）。
- 触发流程是：定位目标消息 -> 打开该消息“更多操作” -> 点击“新聊天中的分支”。

## 安装

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目目录（包含 `manifest.json`）
5. 刷新 ChatGPT 页面

## 使用建议

1. 先点 `扫描`，确认问题栏和计数正确。
2. 对关键回复点 `设为节点`。
3. 需要分叉讨论时点 `开分支`，多节点时先选分支起点。
4. 长对话可切到 `极简`，用 `上/下` 快速巡航。

## 已知限制

- “新聊天中的分支”依赖 ChatGPT 当前页面结构与菜单可见性；若官方 DOM 变更，可能需要更新选择器逻辑。
- 分支本质上是新会话，不是原会话内部真正分叉。
- 扩展不修改 ChatGPT 服务端上下文，只做前端导航和操作辅助。

## 项目结构

- `manifest.json`：Chrome MV3 配置。
- `content.js`：核心逻辑（扫描、导航、节点、分支、状态持久化）。
- `content.css`：UI 样式与动画。
- `lib/interact.min.js`：拖拽能力（interact.js）。
